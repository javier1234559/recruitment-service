# Recruitment Serivce api
Recruitment Serivce api is a project to pratice spring boot

## Features
These services can perform
- Swagger api
- Redis cache server
- Employeer CRUD operations.
- Seeker CRUD operations.
- Job CRUD operations.
- Resume CRUD operations.
- Auth Login api
